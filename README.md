# Stud-Placement-Dashboard
1) Project Title/ Headline:
   Student Placement Analysis Dashboard using Power BI.

2) Short Description/ Purpose:
   This project presents an interactive Student Placement Analysis Dashboard built using Power BI to analyze factors              influencing student placements. The dashboard helps visualize placement status, academic performance, specialization           trends, and salary distribution, enabling better understanding of the patterns that affect student employability.

3) Tech Stack:
   The dashboard was built using the following tools and technologies:
   📊 Power BI – Data visualization and dashboard creation.
   🔄 Power Query – Data cleaning and transformation.
   🧮 DAX (Data Analysis Expressions) – Creating calculated measures and KPIs.
   📁 CSV Dataset – Source file for student placement data.

4) Data Source:
   The dataset used in this project contains student academic and placement records, including information such as:
   📅 SSC Percentage (10th Grade Marks)
   📚 HSC Percentage (12th Grade Marks)
   🎓 Degree Percentage
   📊 Employability Test Score
   💼 Work Experience
   🎯 Specialization
   💰 Salary Offered
   ✅ Placement Status (Placed / Not Placed)

5) Features/ Highlights:
   Key highlights of the dashboard include:
  🎓 Student Overview – Displays total students, placed students, and placement rate.
  📊 Placement Status Analysis – Visual comparison of placed vs not placed students.
  🎯 Specialization Insights – Shows placement distribution across different specializations.
  📈 Academic Performance Analysis – Examines how academic scores affect placement outcomes.
  💰 Salary Analysis – Displays average salary and salary distribution for placed students.
   🎛 Interactive Filters – Users can filter data by gender, specialization, and work experience.
  📊 Data Visualization – Uses charts, KPIs, and graphs to present insights clearly.
